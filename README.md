# ReactJS and Node.js CRUD Application

Tech Stack:
- ReactJS
- NodeJS
- ExpressJS
- GraphQL / Express GraphQL / Apollo Client
- TypeScript
- MySQL 
- TypeOrm

# Run Instruction 
### Clone repo or download zip

### To run front end:

```
cd react-graphql-node
cd client
npm i
npm start
```
### To run back end:

```
Open another terminal, 
cd server
npm run dev
```
